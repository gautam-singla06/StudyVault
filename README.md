# StudyVault

StudyVault is a desktop study management application built with Electron, React, TypeScript, and Vite. It helps students organize subjects, notes, study resources, and learning materials in one place.

## Features

- Create and manage subjects
- Organize notes by subject
- Store study resources
- Fast search functionality
- Clean and distraction-free interface
- Desktop application powered by Electron
- Local data storage

## Tech Stack

- Electron
- React
- TypeScript
- Vite
- Tailwind CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/gautam-singla06/StudyVault.git
```

Install dependencies:

```bash
npm install
```

Run in development mode:

```bash
npm run dev
```

Build the application:

```bash
npm run build
```

Package the desktop application:

```bash
npm run package
```
