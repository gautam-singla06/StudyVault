# 📚 StudyVault

### Your Personal Knowledge Hub for Notes, Resources & Study Management

StudyVault is a modern desktop application designed to help students organize their academic life in one place. Manage subjects, create notes, store learning resources, and keep study materials structured without relying on multiple apps.

## 🚀 Overview

Students often keep notes, PDFs, resources, links, and study material scattered across different folders and applications.

StudyVault solves this problem by providing a dedicated desktop workspace where everything related to learning can be organized, searched, and accessed quickly.

Whether you're preparing for exams, managing multiple subjects, or collecting resources for projects, StudyVault keeps everything centralized.

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/gautam-singla06/StudyVault.git
```

Move into the project:

```bash
cd StudyVault
```

Install dependencies:

```bash
npm install
```

Run development mode:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```

Package desktop application:

```bash
npm run package
```
